💡 Como funciona
O servidor busca a página completa e usa o Mozilla Readability para extrair só o conteúdo do artigo — o mesmo motor que o Firefox usa no modo leitura.

Funciona bem quando o site carrega o artigo no HTML (muitos fazem isso mesmo com paywall).
Se o conteúdo estiver mesmo bloqueado, aparecem links alternativos como o Archive.today.
